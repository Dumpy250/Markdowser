# Markdowser
> A markdown web browser.

## Setup

1. Download one of the [releases](https://github.com/Stone-Red-Code/Markdowser/releases).
1. Execute the `Markdowser-Installer.exe` file.
1. Enter a URL in the address bar.
1. Enjoy!
